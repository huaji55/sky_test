# 盛宴日 (需接取歸巢季第三季節任務)

今年**盛宴節**與愛麗絲夢遊仙境合作，將其帶入Sky，慶祝這場活動吧!
在遇境或雲巢跟隨螃蟹的步伐，來到咖啡廳，並跌進仙境咖啡廳展開一場奇妙冒險!

## 活動時間
- **開始時間**：12月23日下午4點
- **結束時間**：1月13日下午4點
- **持續時間**：21天

## 活動內容

1. - 每日可獲得**5個活動代幣**，此次一樣是燭光形式
     - 每天有2位先祖，幫助他們完成任務可獲得5個活動代幣

2. - 一次性**15個活動代幣**，分佈在活動地圖周圍，可分天慢慢尋找，不會刷新
