# Sky 更新日志 - 0.28.1 (2025年1月31日)

**版号**
- 安卓、华为、iOS、Steam, Switch: 310103
- PS : 310141

## 更新内容：

1. **修正彩染季挑衅艺伎先祖奇怪动画**：修正了可能出现的一些动画问题。
2. **修正二级斗篷颜色不一致**：包括二级青色斗篷的颜色不一致问题。
3. **修正巢穴显示为空的错误**：当玩家在巢穴位置开始游戏时，可能会导致显示为空。
4. **解决编辑水晶无法使用的问题**：在关闭访客权限后，玩家无法使用巢穴中的「编辑」水晶。
