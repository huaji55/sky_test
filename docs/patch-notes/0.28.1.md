# Sky 更新日誌 - 0.28.1 (2025年1月31日)

**版號**
- 安卓、華為、iOS、Steam, Switch: 310103
- PS : 310141

## 更新內容：

1. **修正彩染季挑釁藝伎先祖奇怪動畫**：修正了可能出現的一些動畫問題。
2. **修正二級斗篷顏色不一致**：包括二級青色斗篷的顏色不一致問題。
3. **修正巢穴顯示為空的錯誤**：當玩家在巢穴位置開始遊戲時，可能會導致顯示為空。
4. **解決編輯水晶無法使用的問題**：在關閉訪客權限後，玩家無法使用巢穴中的「編輯」水晶。
